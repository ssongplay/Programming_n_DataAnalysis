# 프로그래밍과 데이터분석

👩🏻‍💻 2021학년도 2학기 수강

### 교과목 개요
- 파이썬을 활용하여 프로그래밍의 기초 개념을 학습한다.
- 파이썬의 자료형, 연산자, 조건문, 반복문, 함수, 클래스를 활용할 수 있으며 파이썬 스타일 코드, 모듈 및 패키지, 예외처리를 이해할 수 있다.

### 교재
데이터 과학을 위한 파이썬 프로그래밍 최성철 한빛아카데미 2019
